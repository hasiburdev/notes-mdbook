# CSS

## Flexbox

### Flex container

**Properties**

- `display:flex`
- `align-items`
- `justify-content`
- `flex-direction`
- `flex-flow`
- `align-content`

* `align-content: stretch` will adjust the height of the children boxes same as the largest height of any children.

### Flex children

**Properties**

- `flex-basis`
- `order`
- `flex-grow`
- `flex-shrink`
